# Functional Prototype

This folder contains a mobile-style React to-do list prototype with a fake backend.

## Run locally

1. Install Node.js 18+.
2. In this folder, run `npm install`.
3. Start the app with `npm run dev`.
4. Open the local URL Vite prints in your browser.

## Notes

- The UI is intentionally framed like a phone screen for laptop use.
- The backend behavior is simulated in the browser; nothing is sent to a server.
